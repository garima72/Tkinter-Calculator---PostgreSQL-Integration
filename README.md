# Tkinter Calculator - PostgreSQL Integration
# 🧮 Tkinter Calculator with PostgreSQL History

A desktop calculator built using **Python Tkinter** with **PostgreSQL database integration** to store and display calculation history. This project demonstrates GUI development along with database connectivity in Python.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear input
- 📜 View calculation history
- 🗄️ Automatic history storage in PostgreSQL
- 🔒 Parameterized SQL queries for secure database operations
- ⚠️ Division by zero handling

---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- PostgreSQL
- psycopg2
